# 🩺 Diabetes Prediction Using Machine Learning

## 📌 Project Overview

This project predicts whether a person is diabetic or not using Machine Learning and Streamlit.

The application takes patient health details as input and provides real-time diabetes prediction with confidence score and healthcare recommendations.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle

---

## 📊 Machine Learning Model

- Logistic Regression
- Feature Scaling
- ROC-AUC Evaluation
- Confusion Matrix

---

## ✨ Features

✅ Real-time diabetes prediction  
✅ Interactive Streamlit web app  
✅ BMI status analysis  
✅ Confidence score  
✅ Healthcare recommendations  
✅ Professional UI dashboard
